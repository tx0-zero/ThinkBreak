# ThinkBreak v0.1.0

## 中文

ThinkBreak 首个开源测试版：AI 在思考，你先休息一下。

- 支持 Codex 和 Claude Code。
- 任务超过 2 秒后打开用户选择的 Chrome 网页。
- 完成、授权请求或安全超时时暂停媒体并返回原窗口。
- 支持媒体/阅读预设、标签页复用、总开关和本地配置。
- “看广告换 Token”只是禁用的玩梗开关，不包含真实广告或 Token 服务。
- 无遥测、无分析 SDK，网址和任务窗口信息只留在本机。

下载包使用 ad-hoc 签名，**未经 Apple 公证**。请阅读 README 中的首次打开、辅助功能和 Chrome Apple Events 权限说明。

## English

The first open-source beta of ThinkBreak: take a break while your AI is thinking.

- Supports Codex and Claude Code.
- Opens user-selected Chrome content when a task runs longer than two seconds.
- Pauses media and restores the originating window on completion, permission request, or safety timeout.
- Includes media/reading presets, reusable tabs, a master switch, and local-only configuration.
- The “Watch ads for Tokens” control is a disabled joke; there are no real ads or token services.
- No telemetry or analytics SDK. URLs and transient task-window identifiers stay on the Mac.

The download is ad-hoc signed and **not Apple-notarized**. Read the README for first-launch, Accessibility, and Chrome Apple Events setup.
