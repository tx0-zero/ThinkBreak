# ThinkBreak v0.1.0

## 中文

ThinkBreak v0.1.0 是一个 Hook-first 开源测试版：Agent 工作时执行用户定义的等待 Recipe，完成或需要用户时返回。

- 支持 Codex 与 Claude Code 的 `UserPromptSubmit`、`PermissionRequest`、`Stop` 生命周期。
- 默认抖音样板使用系统默认浏览器，沿用用户已有登录会话；核心不读取凭证。
- Recipe 可以替换为 B 站、小说、音乐、文章、个人网站或任意本地脚本。
- 默认 2 秒短任务保护、最新任务 Owner、来源窗口恢复和 30 分钟安全超时。
- macOS/Linux 使用 Bash；Windows 提供 PowerShell Dispatcher。
- 无桌面应用、无浏览器扩展、无 MCP、无遥测、无广告 SDK、无真实 Token 奖励。
- “看广告换额度”仅保留为可选网站 Recipe 的玩梗/兴趣收集方向。

## English

ThinkBreak v0.1.0 is a Hook-first open-source test release: run a user-defined waiting Recipe while an Agent works, then return when it finishes or needs attention.

- Supports the `UserPromptSubmit`, `PermissionRequest`, and `Stop` lifecycle events used by Codex and Claude Code integrations.
- The default Douyin example opens the system browser and naturally reuses the user's existing login session; the Core never reads credentials.
- Recipes can be replaced with Bilibili, novels, music, articles, personal websites, or local scripts.
- Includes the two-second short-task guard, newest-task ownership, source-window restoration, and a 30-minute safety timeout.
- Bash is provided for macOS/Linux and PowerShell for Windows.
- No desktop app, browser extension, MCP server, telemetry, ad SDK, or real token rewards.
- “Watch ads for model credits” remains only a joke/interest-collection direction for an optional user website Recipe.
