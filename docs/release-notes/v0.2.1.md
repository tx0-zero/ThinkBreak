# ThinkBreak v0.2.1

## 中文

ThinkBreak v0.2.1 是一次仓库迁移和发布维护更新。

- 项目公开仓库迁移到 `github.com/tx0-zero/ThinkBreak`。
- README、英文 README 和贡献指南中的安装地址已更新。
- Codex 与 Claude Code 插件版本统一为 `0.2.1`。
- Hook-first Dispatcher、Recipes、本地配置格式和运行行为保持不变。
- 不引入桌面应用、常驻服务、MCP、浏览器扩展、遥测、广告或 Token 服务。

macOS/Linux 使用 Bash Dispatcher，Windows 使用 PowerShell Dispatcher。请在安装或更新后重新启动 Codex / Claude Code。

## English

ThinkBreak v0.2.1 is a repository-migration and release-maintenance update.

- Moves the canonical public repository to `github.com/tx0-zero/ThinkBreak`.
- Updates installation links in both READMEs and the contribution guide.
- Aligns the Codex and Claude Code plugin versions at `0.2.1`.
- Keeps the Hook-first Dispatcher, Recipes, local configuration format, and runtime behavior unchanged.
- Adds no desktop app, resident service, MCP server, browser extension, telemetry, advertising, or token service.

macOS/Linux use the Bash Dispatcher; Windows uses the PowerShell Dispatcher. Restart Codex / Claude Code after installing or updating.
